用法：

只需将<script src='(...)imconfig.js'>放入页面底部，</body>之前

暂时只提供宽度、高度的配置，可修改imconfig.js下面的属性:
window.easemobConfig = {
  width: 300//悬浮窗宽度
  , height: 400//悬浮窗高度
  , position: 'bottom'//悬浮窗位置
};
